# Como Configurar o Favicon

Para que o ícone "C" apareça corretamente na aba do navegador usando a mesma fonte da logo:

1. **Copie o arquivo manualmente:**
   - De: `src/assets/images/logo-icon.png`
   - Para: `public/favicon.png`

2. **Ou execute no terminal:**
   ```bash
   cp src/assets/images/logo-icon.png public/favicon.png
   ```

3. **Recarregue o navegador** (F5 ou Ctrl+Shift+R / Cmd+Shift+R)

O favicon agora usará a mesma imagem do ícone "C" que está na logo completa!
